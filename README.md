# Rock Paper Scissors
This is a simple game of Rock, Paper, Scissors. The game is played against the computer.

### Features:
- The game is played in the console and includes 5 rounds.
- A prompt is asked from the user and accordingly the result of the round is determined
- Winner is declared at the end of each round and the score is updated.

### Implementation:
- The Web UI is implemented using HTML & CSS ( flexbox )
- The game is implemented using JavaScript.
